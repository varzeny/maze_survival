### 개요
web game by websocket

### 개발동기
- websocket을 이용한 실시간 웹게임 서비스를 개발해보며 종합적인 백앤드 실력 향상을 도모한다.

### 단계
1. 클라이언트가 브라우저로 보내는 wasd 를 실시간 출력한다.
2. 가로세로 100 크기의 평면(ground)(그라운드)에 캐릭터를 배치하여, 캐릭터를 wasd로 움직인다.
3. 하단에 일종의 상태창 대쉬보드
4. 랜덤미로(알고리즘)
5. 적대 NPC
6. 서바이벌??

### 환경

#### 서버
- 배포 : 로컬서버 or AWS
- OS : 우분투 20.04+
- 언어 : python
- 도구 : uvicorn + fastapi
- DB : MySQL

#### 클라이언트
- 브라우저
